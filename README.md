Уникальные возможности (общие для всех версий)
🌍 Поддержка культур: английские, русские, испанские, немецкие, французские, итальянские имена (и фамилии).

🧑‍🤝‍🧑 Выбор пола: мужские, женские или любые.

🔢 Гибкое количество: генерация одного или нескольких имён за раз.

🧩 Режимы генерации:

Только имя — классическое имя.

Имя + фамилия — полное имя.

Никнейм — креативное сочетание (например, «Храбрый Лис»).

📂 Экспорт результатов в текстовый файл (names.txt) или CSV.

🎲 Случайность с контролем — можно задать seed для воспроизводимости.

📝 Настраиваемые списки — возможность добавлять свои имена через JSON-файл (в большинстве версий).

🖥️ Интерфейсы:

Командная строка (CLI) с богатыми опциями.

Графический интерфейс (GUI) в Python и C#.

Веб-интерфейс в JavaScript и PHP.
